chooseOne()
{
	if 維修機台數量沒超過上限
	{
		考慮維修可否最加快完成
		if 出現兩種以上最佳解
		{
			選擇之中不用維修的
			if 都不用維修
			{
				選擇之中目前良率低的
                        }
			if 都要維修
			{
				選擇維修時間最短的
				
				if 維修時間相同
				{
					選擇良率低的
				}
			}
		}
	}
